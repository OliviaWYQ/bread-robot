# 开发环境搭建和调试计划

项目总周期: 6-8周，分四阶段进行

---

## 阶段一：环境搭建与硬件集成（第1-2周）

### 目标
完成LeRobot环境部署，所有硬件驱动调试通，传感器数据能正常读取。

### 任务清单

| 序号 | 任务 | 预计耗时 | 交付 |
|------|------|----------|------|
| 1.1 |  Ubuntu 22.04 系统安装配置，NVIDIA驱动更新 | 1天 | 系统就绪 |
| 1.2 | 克隆 LeRobot 源码，按照官方文档安装依赖 | 1天 | 可运行 `lerobot.control` 测试 |
| 1.3 | 机械臂驱动安装和基本运动测试 | 1-2天 | 可以遥操作控制机械臂运动 |
| 1.4 | 所有相机安装标定，内参外参标定 | 1-2天 | LeRobot 可以正常录制带多相机数据的轨迹 |
| 1.5 | 电子秤驱动开发，串口/蓝牙通信测试 | 1天 | 可以Python实时读取重量数据 |
| 1.6 | 力传感器安装调试（如果外接） | 1-2天 | 可以读取六维力数据并入LeRobot观测 |
| 1.7 | 工作站布局固定，所有工位标定 | 1天 | 各个物料工位坐标确定 |

### 验收标准
- 可以成功录制一条示教轨迹，包含关节位置、图像、力、重量数据
- 时间线对齐正确，没有丢包

---

## 阶段二：数据采集（第3-4周）

### 目标
完成所有子任务的示教数据采集，清理上传到数据集。

### 任务清单

| 任务 | 预计耗时 | 说明 |
|------|----------|------|
| 制定采集脚本，分任务创建数据集 | 0.5天 | 按照子任务拆分数据集 |
| 配料称量数据采集（粉/水/盐/糖/酵母） | 3-4天 | 每个任务多次采集，覆盖不同初始量 |
| 揉面数据采集 | 2-3天 | 不同硬度面团，注意安全防止卡住 |
| 整形和转移数据采集 | 1-2天 | 发酵箱和烤箱操作 |
| 开门关门数据采集 | 1天 | 注意力控，不要硬撞 |
| 数据清洗过滤，上传Hub | 0.5天 | 删除失败轨迹，版本管理 |

### 注意事项
- 面团操作会弄脏相机，定期清理镜头保证图像清晰
- 高温操作（烤箱）注意安全，示教时断电机械臂防止误动
- 每天采集完及时备份数据

---

## 阶段三：模型训练与开发（第5周）

### 目标
完成各个子任务模型训练，开发任务调度流程。

### 任务清单

| 任务 | 预计耗时 | 说明 |
|------|----------|------|
| ACT baseline 模型训练（各个子任务） | 2-3天 | 从预训练权重开始全参数微调 |
| 验证评估：每个任务成功率和精度测试 | 1天 | 统计成功次数，测量称量精度 |
| 如果精度不满足，切换Diffusion Policy训练对比 | 1-2天 | 对比两种算法效果 |
| 开发任务调度状态机 | 1-2天 | 把各个子任务串成完整流程 |
| 闭环称量逻辑开发 | 1天 | 增量倾倒+重量反馈逻辑 |

### 验收标准
- 单个子任务成功率达到 80% 以上
- 称量精度满足 ±1g 要求
- 完整流程可以从配料走到烘烤前

---

## 阶段四：联调测试与优化（第6-8周）

### 目标
全流程联调，解决问题，优化成功率，交付可用MVP。

### 任务清单

| 阶段 | 任务 | 耗时 |
|------|------|------|
| 分阶段测试 | 配料称量全流程测试，统计精度和成功率 | 1-2天 |
| 分阶段测试 | 揉面整形流程测试，观察面团形变处理 | 2-3天 |
| 分阶段测试 | 发酵箱/烤箱开门存取测试 | 1-2天 |
| 全流程联调 | 从0开始完整跑通一次面包制作 | 1天 |
| 问题收集 | 记录失败场景，统计各个环节失败率 | 1天 |
| 数据补充 | 针对失败场景补充采集更多数据，finetune模型 | 2-3天 |
| 优化迭代 | 根据测试结果调整算法或硬件设计 | 2-3天 |
| 最终验收 | 连续三次完整跑通，产出可食用面包 | 1天 |

### 风险应对计划

| 风险 | 应对方案 |
|------|----------|
| 柔性揉面效果差，模型泛化不够 | 补充更多不同面团数据，加入力反馈作为输入，换Diffusion Policy |
| 称量精度不够 | 增加更多不同起始重量数据，优化增量倾倒策略，检查标定误差 |
| 开门失败 | 增加力控阈值，碰到阻力停止重试，优化示教轨迹 |
| 时间不够 | 砍掉全自动化，发酵烘烤存取改为人机协作，优先保证前面称量揉面功能 |

## 工程量估计

- 总人天: ~30-40 人天（一人开发）
- 主要工作量: 数据采集（~10天），调试验证（~10天）
- 算法改动量: 不大，主要是LeRobot基础上扩展输入输出和调度层
