# 自动面包机器人任务拆解

本文档详细拆解完整面包制作流程为LeRobot可执行的任务模块。

## 整体流程

```
原料准备 → 混合 → 揉面 → 一次发酵 → 整形 → 二次发酵 → 预热烤箱 → 烘烤 → 冷却 → 完成
```

## 详细任务模块拆解

### 预处理（用户操作）

| 任务 | 执行人 | 说明 |
|-----|-------|-----|
| 清洁所有工具 | 用户 | 保证食品安全 |
| 称量各种原料 | 用户 | MVP阶段由用户完成称量，装入对应容器 |
| 摆放所有容器到指定位置 | 用户 | 机器人按固定位置取料 |
| 将揉面盆放在固定台面上 | 用户 | |
| 选择配方启动程序 | 用户 | |

---

## 模块1：原料投放

### 任务1.1 - 视觉定位原料容器
- **输入：** 相机图像
- **输出：** 各个容器的位姿
- **动作：**
  1. 相机拍摄整个原料区
  2. LeRobot视觉模型检测识别各个容器
  3. 计算每个容器的抓取位置和倾倒角度
- **LeRobot支持：** 原生支持物体检测

### 任务1.2 - 抓取面粉容器
- **动作：**
  1. 机械臂移动到面粉容器上方
  2. 下降到抓取位置
  3. 夹爪闭合夹紧容器
  4. 抬起容器
- **关键点：** 夹爪力度控制，避免夹碎容器

### 任务1.3 - 倾倒面粉到揉面盆
- **动作：**
  1. 移动容器到揉面盆上方
  2. 缓慢倾斜容器控制流速
  3. 轻敲容器壁帮助面粉全部倒出
  4. 保持倾斜姿态数秒
- **关键点：** 控制角度避免面粉撒出

### 任务1.4 - 面粉容器归位
- **动作：** 将容器放回原料区原位置

### 任务1.5 - 依次添加其他原料
- 重复上述步骤添加水、酵母、盐

---

## 模块2：揉面

### 任务2.1 - 更换末端执行器
- **动作：**
  1. 将夹爪放到自动换头架
  2. 松开夹爪，移动机械臂离开
  3. 移动到揉面勾位置，对准接口
  4. 夹紧固定揉面勾
- **需求：** 需要定制自动换头机构

### 任务2.2 - 初步混合原料
- **动作：**
  1. 揉面勾伸入揉面盆
  2. 缓慢搅拌，将干粉和水混合成絮状
  3. 刮干净盆壁混合均匀
- **关键点：** 控制深度，避免刮伤揉面盆

### 任务2.3 - 揉面循环（重复10-15次）
- **动作序列（一次循环）：**
  1. 对折面团一端到另一端
  2. 用揉面勾向下按压面团
  3. 旋转面团一定角度
  4. 重复
- **关键点：** 压力控制，既要揉到又不挤压过度

### 任务2.4 - 揉面完成判断
- **MVP方案：** 基于固定次数和时间
- **进阶方案：** 视觉判断面团光滑度

### 任务2.5 - 面团收圆
- **动作：** 将面团整理成球形，放在发酵盆中心

---

## 模块3：第一次发酵

### 任务3.1 - 移动发酵盆到发酵位置
- **动作：**
  1. 更换回夹爪末端
  2. 抓取发酵盆边缘
  3. 平稳移动到保温发酵区
  4. 放下固定
- **关键点：** 平稳移动避免面团变形

### 任务3.2 - 等待发酵
- 计时1-1.5小时（根据配方）
- 保持环境温度（如果有保温装置）

### 任务3.3 - 发酵完成检查（可选）
- 视觉检测体积膨胀倍数
- MVP跳过，基于时间判断

---

## 模块4：整形

### 任务4.1 - 排气
- **动作：** 按压面团排出大的气泡

### 任务4.2 - 滚圆
- **动作：**
  1. 将面团在操作台上滚动
  2. 收紧表面形成光滑圆球
- **关键点：** 压力控制

### 任务4.3 - 转移到发酵篮
- **动作：** 将面团放入发酵篮，表皮向上

---

## 模块5：第二次发酵

### 任务5.1 - 移动发酵篮到发酵位置
- 和一次发酵类似，平稳移动

### 任务5.2 - 等待二次发酵
- 计时30-60分钟

---

## 模块6：烘烤准备

### 任务6.1 - 烤箱预热检查
- **动作：**
  1. 读取烤箱温度传感器
  2. 判断是否达到预定温度
  3. 如果未达到，等待（智能烤箱可以发送预热指令）
- **MVP：** 用户提前预热好，机器人只做检查

### 任务6.2 - 打开烤箱门
- **动作：**
  1. 定位烤箱门把手
  2. 夹取门把手
  3. 缓慢匀速拉门打开
  4. 开门到位后保持/固定
- **关键点：** 力控，避免弹开过猛

### 任务6.3 - 放入面团
- **动作：**
  1. 夹取烤盘/发酵篮
  2. 伸入烤箱放到预定烤架位置
  3. 放下后机械臂退出烤箱
- **关键点：** 注意高温防护

### 任务6.4 - 关闭烤箱门
- **动作：** 缓慢推关门到位

### 任务6.5 - 开始烘烤计时
- 记录开始时间，设定倒计时

---

## 模块7：烘烤完成取出

### 任务7.1 - 烘烤时间到处理
- 等待烤箱降温（开门前等待数分钟让热浪散出）

### 任务7.2 - 打开烤箱门
- 同步骤6.2

### 任务7.3 - 取出面包烤盘
- **动作：**
  1. 机械臂伸入夹取烤盘
  2. 平稳移出烤箱
  3. 放到冷却架位置
- **关键点：** 防烫设计，末端需要隔热

### 任务7.4 - 关闭烤箱门
- 如果不再使用，关闭烤箱电源

---

## 模块8：冷却完成

### 任务8.1 - 冷却计时
- 等待15-30分钟让面包内部成熟

### 任务8.2 - 通知用户
- 发送APP通知，面包做好可以取出

---

## 模块9：异常处理

### 9.1 碰撞检测
- 当机械臂检测到意外碰撞，立即停止所有动作
- 发送报警通知用户

### 9.2 温度异常
- 烤箱温度超过安全阈值 → 停止操作，开门降温，报警
- 温度达不到预定 → 报警提醒用户检查

### 9.3 物体定位失败
- 重新拍摄识别三次
- 如果仍然失败 → 暂停，通知用户确认物料摆放

### 9.4 紧急停止
- 物理急停按钮优先级最高，任何时候按下立即断电停止

---

## 任务模块接口定义

每个任务模块输出：
- **状态：** SUCCESS / FAILURE
- **失败原因：** 错误代码
- **数据：** 输出位姿、检测结果等

下一模块仅在上一模块SUCCESS后启动。
