# 面包机器人项目（Bread Robot MVP）

基于LeRobot平台开发自动做面包的家务机器人MVP

## 项目目标
6-8周交付可以完整制作一个面包的原型机

## 团队分工

| 角色 | 姓名 | 职责 |
|------|------|------|
| 📊 BizClaw | 商业市场AI助手 | 商业分析、方案框架、市场测算、商业模式 |
| 🎨 ProductClaw | 产品AI助手 | 产品定义、用户痛点、MVP范围、验收标准 |
| 🔧 TechClaw | 技术AI助手 | 技术架构、硬件改造、算法开发、工程落地 |
| 👤 mac小爪 | 项目发起人 | 协调推进、硬件采购、实际测试 |

## 项目阶段

### 阶段一：需求定义与方案设计
- [ ] 产品定义：MVP范围和验收标准（ProductClaw）
- [ ] 商业分析：市场定位、成本测算（BizClaw）
- [ ] 技术方案：改造方案、工程量评估（TechClaw）

### 阶段二：硬件改造
- [ ] 末端执行器定制设计
- [ ] 传感器增补安装
- [ ] 工作站布局搭建

### 阶段三：软件开发与数据采集
- [ ] 环境配置与代码适配
- [ ] 人工示范数据采集
- [ ] 模型微调

### 阶段四：联调测试
- [ ] 分阶段测试
- [ ] 全流程跑通
- [ ] 问题修复与优化
