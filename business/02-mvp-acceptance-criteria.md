# 做面包机器人 MVP - 验收标准与功能清单

## MVP 目标

验证**全自动制作基础白吐司/餐包**的可行性，让用户放入原材料后，无需人工干预即可得到成品面包。

---

## 必须实现功能清单

### 1. 原材料管理
- [ ] 支持至少 4 种基础原材料储存：高筋面粉、干酵母、盐、水
- [ ] 原材料余量检测，不足时提醒用户补充
- [ ] 酵母冷藏功能（维持 4-8℃）

### 2. 自动称量
- [ ] 按配方精度要求称量每种原材料
  - 面粉精度 ±5g
  - 盐/酵母精度 ±1g
  - 水精度 ±2g
- [ ] 称量完成后自动倒入揉面盆
- [ ] 支持至少 450g 吐司配方的总容量

### 3. 自动揉面
- [ ] 能够完成干粉混合→加水成团→揉面出膜完整流程
- [ ] 能揉出符合吐司要求的完全阶段手套膜
- [ ] 揉面过程温度控制，面团最终温度不超过 28℃
- [ ] 揉面完成自动整理成团

### 4. 可控发酵
- [ ] 一次发酵：维持 26℃ ±2℃，湿度 75% ±5%
- [ ] 能够通过体积/戳洞检测判断发酵完成
- [ ] 发酵完成自动进入下一环节
- [ ] 二次发酵：维持 35℃ ±2℃，湿度 80% ±5%

### 5. 整形入模
- [ ] 能够完成排气→滚圆→松弛→擀卷→入模完整整形流程（针对吐司）
- [ ] 适配标准 450g 吐司盒

### 6. 烘烤控制
- [ ] 支持与家用烤箱联动控制（或自带烤箱模块）
- [ ] 自动预热到设定温度
- [ ] 自动入炉，设置烘烤时间温度
- [ ] 烘烤完成自动出炉

### 7. 成品取出
- [ ] 烘烤完成后自动将面包从模具取出放到冷却架
- [ ] 推送通知告知用户制作完成

### 8. 用户交互
- [ ] 手机 App / 屏幕选择预设配方（基础白吐司、原味餐包）
- [ ] 制作过程实时进度展示
- [ ] 异常状态报警（原材料不足、称量失败、温度异常等）
- [ ] 紧急停止按钮

### 9. 清洁维护
- [ ] 揉面工具可拆卸清洗
- [ ] 提示用户定期清洁维护

---

## MVP 验收标准

### 成功标准
1. **全自动流程验证**：用户放入原材料后，从称量到出炉全程无需人工干预，能够完整产出成品面包
2. **成品质量要求**：
   - 体积达到正常水平（450g 吐司盒填充率 80%-90%）
   - 组织气孔均匀，能揉出完全阶段手套膜
   - 烤色均匀，无烤焦或未熟情况
   - 口感正常，无盐多盐少、发酵过度等重大缺陷
3. **成功率要求**：连续 3 次成功制作出符合要求的吐司，成功率 ≥ 85%
4. **安全性要求**：
   - 高温部分有防护，不会造成烫伤
   - 运动部件有安全检测，夹手能自动停止
   - 用电安全符合家用标准

### MVP 不包含的功能 (Postpone to v2)
- [ ] 多种复杂配方（全麦、可颂、丹麦等）
- [ ] 自动添加黄油等配料
- [ ] 蒸汽喷射系统（欧式面包需要）
- [ ] 自动清洁揉面盆
- [ ] 多个面包同时制作
- [ ] 自定义配方编辑（保留预设配方）
- [ ] 口味推荐系统
